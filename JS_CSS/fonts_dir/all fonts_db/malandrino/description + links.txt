MALANDRINO description:

—————————————————————————

Malandrino is a Decorative Font.
In this font the playful rounded Serifs pop out from the geometric shapes, giving a nice lively feeling.
It could be called a Geometric Serif Font with remarkable Display characteristics, but it also fit in long text blocks.
The height difference between Uppercase and Smallcase make it more suitable for Typography than Web.

Malandrino Family:
Malandrino Regular
Malandrino Bold
Malandrino Light
